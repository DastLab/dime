DiMe Software Repository

This page contains the DiMe and the datasets used to accomplish the experiments presented in the paper "Mining Relaxed Functional Dependencies from Data", by Loredana Caruccio, Vincenzo Deufemia, and Giuseppe Polese, submitted for publication to Data Mining and Knowledge Discovery.

This link allows to download a compressed archive (zip format) containing the following files:

  • a jar file concerning the implementation of DiMe (DiMe.jar)
  • the datasets used for the evaluation
  • a readme file explaining how to use the software for replicating the experiments (Readme.txt).
